# Fairy Rescue
Game Link : https://play.unity.com/en/games/7e082e08-7111-459e-a52c-97c65530ed7e/my-fairy-rescue

A 2D Unity platformer where players explore magical worlds, collect keys, and unlock portals to progress through levels.

## Overview
Fairy Rescue is a multi-level game designed to introduce players to core platforming mechanics such as movement, jumping, and interaction with the environment. Each level presents new challenges while guiding the player toward the ultimate goal of unlocking the next world.

## Features
- Smooth player movement and jumping mechanics using Unity physics
- Coin collection system and interactive environment elements
- Portal system for level progression
- Multi-level world design with increasing difficulty
- Restart and life system for gameplay feedback

## Tech Stack
- Unity (C#)
- 2D Physics System
- Sprite Animation & Animator
- Collision Detection & Game Logic Systems
